# GLSL Custom
Rozszerzenie do Visual Studio Code dodające wsparcie dla shaderów GLSL, między innymi dodaje: kolory składni, podstawowe wbudowane podpowiadanie kodu (toporne), sprwadzanie błędów - też wbudowane. Stworzyłem to z myślą o innych osobach piszących w GLSL żeby nie męczyli się 2 godziny nad literówką w shaderze.


> 🤖 **Project Note:** This extension was built with the assistance of AI, combining smart automation with an engineering approach to GLSL development
